# Math 240
## Intro to Computational Math

### Course Description
This course provides a survey of computational techniques and methods rooted in mathematics and computing. The primary focus is on constructing computer programs to implement mathematical techniques and applying them to "real-life" problems in economics, physics, and the life sciences.

### Key Topics Covered
* **Numerical Differentiation & Integration:** Calculating derivatives and integrals using computational algorithms.
* **Differential Equations:** Numerical solutions to ordinary differential equations (ODEs).
* **Root Finding:** Solving non-linear equations in one variable.
* **Classification Methods:** Implementing computational models to categorize data.

### Technologies Used
* **Language:** Python
* **Environment:** Jupyter Notebooks (`.ipynb`)
* **Libraries:** NumPy, Matplotlib, SciPy
